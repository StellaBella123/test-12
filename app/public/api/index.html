API Documentation

List of APIs:

  • /api/rap
  • /api/joindate
  • /api/latestitems
  • /api/usernames

How to use them: 

        The vast majority of API's will require either a userId, or a username parameter.
        You can use either one (userId is faster), but only one should be used at a time.

        The API's will always be returned in JSON - an error will be indicated
        by a status code of 500, and an "error" parameter (along with the error message and type)
        will be returned.
